# SQL CRM Analysis

Developed a structured data model using SQL (CTEs, Window Functions) to analyze user behavior and email marketing effectiveness. Calculated conversion funnels and implemented a market ranking system, identifying the top 10 most efficient markets.

## Key Insights & Deliverables
- Built end-to-end user conversion funnels to detect drop-off stages.
- Ranked international markets based on custom commercial KPIs.
- Optimized query performance using advanced analytical functions.
